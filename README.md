# Sentiment Analyzer

> Paste any text. Know exactly how it feels.

A lightweight, single-file sentiment analysis tool powered by Groq that classifies text into emotional states — happy, sad, mad, or curious — in real time.

## What It Does

Paste a review, sentence, or paragraph and get instant emotional classification. Built for speed — no backend, no setup, just open and use.

## Features

- Classifies sentiment into 4 emotional states: **Happy, Sad, Mad, Curious**
- Powered by Groq for near-instant inference
- Single HTML file — zero dependencies, zero setup
- Works on any text: reviews, feedback, messages, social posts

## Usage

1. Clone the repo
2. Open `sentiment-analyzer-groq.html` in your browser
3. Paste your text
4. Get instant results

```bash
git clone https://github.com/Marwan7777777/sentiment-analyzer-in-1-file
cd sentiment-analyzer-in-1-file
open sentiment-analyzer-groq.html
```
